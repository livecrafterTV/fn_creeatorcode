What Is A Creator Code?

A Creator Code is a free way to support YouTuber/Twitch Streamers and other Creator when buying VBucks

You want to donate to your favorite Creator and buy VBucks, but you dont have the money? Then the answer is Fortnite's Creator Code. Prices are the same, but in addition you support a Creator. The Creator gets 10% of the money you paid VBucks with!

How To Use A Creator Code


You can redeem the Creator Code within Fortnite. You need to enter the Battle Royale Mode of Fortnite. Afterwards you can go to the Shop and in the top right corner you should be able to see the "Support a Creator" button. Press it and then enter the Code.
